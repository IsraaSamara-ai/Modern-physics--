📖 وصف المشروع | Project Description
تطبيق تعليمي تفاعلي مُصمَّم لتوضيح مفاهيم الفيزياء الحديثة المتعلقة بالطبيعة الجسيمية للضوء. يُقدِّم التطبيق الشروحات والمحاكيات والرسوم المتحركة والمنحنيات البيانية التفاعلية بشكل مبسَّط يُناسب طالباً لا يملك خبرة مسبقة بهذه المفاهيم.

An interactive educational application designed to illustrate modern physics concepts related to the particle nature of light (Scientific Stream). The app provides explanations, simulations, animations, and interactive graphs in a simplified manner suitable for beginners.
